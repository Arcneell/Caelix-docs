# Release notes — v2.2.16

**Date**: July 2026
**Image**: `ghcr.io/arcneell/caelix:2.2.16` (also `:latest`)

> Continuous-audit batch: closes a **secret disclosure** to the technicien role, hardens
> several admin inputs (flag injection / traversal / bind), ends OOMs on large imports, adds
> the last engine arithmetic guard, and fixes stable keys on a shared component.

---

## Security

- **HIGH — secret disclosure to technicians.** `GET /api/app-stacks/{name}` returned each
  service's `env` block (DB passwords, API keys) and full `config` to any authenticated user.
  The detailed view is now **admin-only** (non-admins get the sanitized list, like `/state`).
- **Hardening (admin):** `POST /api/networks/create` rejects `driver/subnet/gateway` values
  starting with `-` (docker flag injection) and validates the name; domains `target_service`
  validated as a docker name (a `../…` can't write outside its dir); template deploy validates
  `image`/`name` (flag injection); app-stack add-service applies `_validate_stack_bind_path`
  (no arbitrary host-path binds), matching the deploy path.

## Backend — ends OOM on uploads

Image `import` (2 GB) and backup `restore` (500 MB) buffered the whole request body in RAM
before writing. They now **stream** the request to a temp file in chunks, enforcing the cap
**incrementally** (413 as soon as it's exceeded) — no memory spike that could OOM the console.

## Engine

- `autoscale_up_threshold` re-read on the "critical metric" path (`autoscale.sh`) was the last
  unguarded read → `^[0-9]+$` guard added (no more aborted autoscale pass).

## Frontend

- `ArrayField` (shared container/service wizard component) used the array index as `:key` on
  splice-removable rows → deleting a middle row mis-bound. Now keyed by a **stable identity**
  (WeakMap on the row object), with no payload change.

## Validation

Full suite (pytest + bats + shell checks + frontend build) green, then the `2.2.16` image was
deployed to the live 3-node cluster (etcd quorum, floating VIP, etcd auth verified).
