# Release Notes — v1.4.2

**Date**: June 2026  
**Image**: `ghcr.io/arcneell/caelix:1.4.2` (also `:latest`)

---

## Summary

v1.4.2 makes the install **turnkey for the proxy and autoscale**: the installer now lays down the missing dependency.

## Change

- **`socat` installed automatically** by `install.sh` (apt / dnf / apk), with `setcap cap_net_bind_service` (best-effort) so it can listen on `:80`/`:443` even outside systemd.

  `socat` is required by the **global proxy** (service/domain publishing) and by the **autoscale load balancers**. Previously it had to be installed by hand or those features failed silently.

After this release, `install.sh --with-systemd` yields a directly usable environment: web console **and** proxy/autoscale ready the moment you enable them.

---

## Upgrade

The `:latest` tag points to this version — the usual install command pulls the fix.
