# Release notes — v2.1.2

**Date**: June 2026  
**Image**: `ghcr.io/arcneell/caelix:2.1.2` (also `:latest`)

> A small usability release: interface actions now give clear visual feedback
> (animated refresh buttons, explicit prune). No engine, API, or configuration
> changes — a drop-in update from 2.1.1.

---

## Action feedback

- **Refresh buttons** (Containers, Images, Networks, Volumes, Stacks, System): the
  icon **spins** while loading and the button is **disabled** for the duration of the
  request. The click is acknowledged immediately, and you can no longer click
  repeatedly without knowing whether anything is happening.
- **Image prune** now targets the **focused node** (like `pull` and `build`) instead
  of an unintended daemon in cluster mode — which made it look like a no-op. The
  confirmation also reports the **reclaimed space** (e.g. "Unused images removed —
  0B"), so the action is acknowledged even when there was nothing to remove.

---

!!! note "Upgrade"
    Update the image to `ghcr.io/arcneell/caelix:2.1.2` and recreate the console
    container on each node. No data or configuration migration is required from 2.1.1.
