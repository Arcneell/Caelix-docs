# Notes de release — v2.1.3

**Date** : juin 2026  
**Image** : `ghcr.io/arcneell/caelix:2.1.3` (également `:latest`)

> Correctif d'interface : le tableau de bord affiche de nouveau les incidents
> récents. Aucun changement de moteur, d'API ni de configuration — mise à jour sans
> rupture depuis la 2.1.2.

---

## Corrections

- **Tableau de bord — « Incidents récents »** : l'encart restait toujours vide.
  L'API renvoie les incidents sous forme de lignes de log brutes
  (`horodatage | sévérité | service | événement | détail`) ; le tableau de bord les
  traitait comme des objets, donc tous les champs étaient vides. Les lignes sont
  désormais correctement analysées, affichées **du plus récent au plus ancien**, avec
  l'horodatage formaté dans le fuseau horaire de l'utilisateur.

---

    n'est nécessaire depuis la 2.1.2.
