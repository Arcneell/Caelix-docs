# Notes de release — v1.4.2

**Date** : juin 2026  
**Image** : `ghcr.io/arcneell/caelix:1.4.2` (également `:latest`)

---

## Résumé

La v1.4.2 rend l'installation **clé en main pour le proxy et l'autoscale** : l'installeur pose désormais la dépendance manquante.

## Changement

- **`socat` installé automatiquement** par `install.sh` (apt / dnf / apk), avec `setcap cap_net_bind_service` (best-effort) pour autoriser l'écoute sur `:80`/`:443` même hors systemd.
  
  `socat` est requis par le **proxy global** (publication de services / domaines) et par les **load-balancers d'autoscale**. Avant, il fallait l'installer à la main, sinon ces fonctions échouaient silencieusement.

Après cette release, un `install.sh --with-systemd` fournit un environnement directement utilisable : console web **et** proxy/autoscale opérationnels dès qu'on les active.

---

## Mise à jour

L'image `:latest` pointe sur cette version — la commande d'installation habituelle récupère le correctif.
